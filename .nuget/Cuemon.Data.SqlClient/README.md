## About

An open-source family of assemblies (MIT license) that targets and complements the Microsoft .NET platform (.NET 6, .NET 5, .NET Core 3.1, .NET Standard 2, Universal Windows Platform and .NET Framework 4.6.1 and newer) by providing vast ways of possibilities for all breeds of coders, programmers, developers and the likes thereof.

It is, by heart, free, flexible and built to extend and boost your agile codebelt.

## **Cuemon.Data.SqlClient** for .NET

The `Cuemon.Data.SqlClient` namespace contains types that provide ways for developers to work with Microsoft SQL Server integrations. The namespace is an addition to the `System.Data.SqlClient` namespace.

More documentation available at [Cuemon for .NET documentation](https://docs.cuemon.net/api/dotnet/Cuemon.Data.SqlClient.html).
